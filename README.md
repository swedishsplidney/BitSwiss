# **BitSwiss**
### What is BitSwiss?
BitSwiss is a free, open source and customizable tool to put all kinds of information onto any USB drive, SD card, maybe even a floppy disk if you feel so inclined.

Have you ever been out and about doing all kinds of digital shenanigans and wished that you had that one documentation? or maybe you're on an airplane designing circuit boards and you just can't remember what that one symbol means. That is why I am creating this. Eventually, I plan on adding all sorts of info, including but not limited to wikipedia entries, stackoverflow libraries, circuit diagram cheat sheets, and more.

It will also ship with a reader and editor so you can read and edit anything from any computer.
